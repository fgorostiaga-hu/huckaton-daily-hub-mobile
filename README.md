# Huckathon: Daily Hub

Daily Hub is a calendar experience for Humand that centralizes the most relevant events, reminders, and employee moments in one place.

The goal is to make planning easier by bringing together birthdays, holidays, time off, company events, wall news, performance deadlines, and other key reminders into a single, actionable calendar hub.

## Overview

This project explores a unified calendar inside Humand / HuGo that helps employees quickly understand **what matters today, this week, and this month**.

Instead of spreading information across multiple modules, Daily Hub creates one central place to view and manage:

- Employee birthdays
- Holidays
- Personal vacations / time off
- Company events
- Relevant wall posts / news
- Performance review deadlines
- Climate survey deadlines

## MVP Scope

The current MVP focuses on delivering a strong demo experience with:

- Monthly and weekly calendar views
- A **Daily Hub** home screen with a summary of the day
- Event filtering by type
- Bubble-style notifications
- Mobile-first UX
- Visual design aligned with the Humand / HuGo aesthetic

## Why it matters

Most HR platforms offer fragmented experiences: absences in one place, announcements in another, tasks elsewhere.

Daily Hub aims to solve that by providing:

- Better visibility of upcoming events
- Faster access to what needs action
- A more contextual employee experience
- A clearer daily planning flow

## Design Principles

- Mobile-first experience
- Clean, low-friction UI
- Humand-inspired visual language
- High-priority events should be easy to spot
- Actionable events with clear context

## Tech

- React
- Tailwind CSS
- Mobile-first prototype approach

## Future Vision

The broader vision includes:

- 1:1 meetings and calls
- Rotative schedules
- Onboarding tasks
- Segmentation-based filters
- Admin analytics dashboard
- Richer notification system (push, email, in-app)

## Benchmark Inspiration

This project was inspired by gaps identified in tools such as:

- Factorial
- Workvivo
- PeopleForce
- BambooHR
- HiBob

While many of them support absences or events, none of them provide a truly unified, employee-centered calendar hub.
